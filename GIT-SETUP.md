# Git-Setup für Equora v56.44

## 1. Projektordner in VS Code öffnen
Öffne den Ordner `Equora Starter v56.44`.

## 2. Git prüfen
```bash
git --version
```

## 3. Repository initialisieren
```bash
git init
```

## 4. Dateien aufnehmen
```bash
git add .
```

## 5. Commit erstellen
```bash
git commit -m "Equora v56.44"
```

## 6. Remote verbinden und pushen
```bash
git remote add origin <DEIN-REPO-URL>
git branch -M main
git push -u origin main
```
